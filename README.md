# Maze Solver
Implementing dijkstra's algorithm to find and visualize the most efficient solution to a random / customizable maze.


#### Getting Started
1. make sure node and npm installed on machine (node -v and npm -v)
2. clone repo to local
3. cd to project
4. npm install
5. npm run dev
